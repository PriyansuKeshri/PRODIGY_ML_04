# PRODIGY_ML_04  
## Hand Gesture Recognition using Machine Learning

This project is part of my **Machine Learning Internship at Prodigy InfoTech**.

The objective of this task is to build a **hand gesture recognition model** capable of identifying different hand gestures from image data.

---

## 📌 Problem Statement

Develop a machine learning model that can classify different hand gestures from images using computer vision techniques.

---

## ⚙️ Technologies Used

- Python
- OpenCV
- NumPy
- Scikit-learn

---

## 📊 Machine Learning Model

Support Vector Machine (SVM)

The model was trained on processed gesture images to recognize different hand gestures.

---

## 🖼 Image Processing Steps

1. Load gesture images from dataset  
2. Convert images to **grayscale**  
3. Resize images to **64×64**  
4. Flatten images into feature vectors  
5. Train SVM classifier for gesture recognition
